မြန်မာ 2D ဆိုတာ ဘာလဲ? သမိုင်းနဲ့ အခြေခံ လမ်းညွှန်

ကံကြမ္မာ & ဂဏန်း ယဉ်ကျေးမှု · ဖတ်ရန် ၄ မိနစ်

မြန်မာနိုင်ငံမှာ 2D (နှစ်လုံး) ဆိုတာ လူသိအများဆုံး ဂဏန်း ယဉ်ကျေးမှု တစ်ခု ဖြစ်ပါတယ်။ နေ့စဉ်လိုလို လူအများစု စိတ်ဝင်စားကြပေမယ့်၊ အသစ်စ လူတွေအတွက်တော့ "2D က ဘယ်လို ဂဏန်းထွက်လဲ" ဆိုတာ ရှင်းရှင်းလင်းလင်း မသိသေးတာ ရှိနိုင်ပါတယ်။ ဒီဆောင်းပါးမှာ အခြေခံကနေ ရှင်းပြပေးပါမယ်။

2D ဆိုတာ

2D ဆိုတာ နှစ်လုံးတွဲ ဂဏန်း (၀၀ ကနေ ၉၉ အထိ) ကို ဆိုလိုပါတယ်။ ဂဏန်းဟာ နေ့စဉ် သတ်မှတ်ထားတဲ့ အချိန်တွေမှာ ထွက်ပေါ်လာပြီး၊ မြန်မာ လူ့အဖွဲ့အစည်းမှာ ကြာရှည်စွာ တည်ရှိနေတဲ့ ယဉ်ကျေးမှု တစ်ခု ဖြစ်ပါတယ်။

ဂဏန်း ဘယ်လို ထွက်လဲ

မြန်မာ 2D ဂဏန်းဟာ နိုင်ငံတကာ စတော့ဈေးကွက် (Stock Market) ရဲ့ ကိန်းဂဏန်းတွေကို အခြေခံပြီး တွက်ချက်လေ့ ရှိပါတယ်။ အဓိက အချက် နှစ်ခု ရှိပါတယ်—

  • Set — အရောင်းအဝယ် ညွှန်းကိန်း
  • Value — ရောင်းဝယ်မှု တန်ဖိုး

ဒီ နှစ်ခုရဲ့ နောက်ဆုံး ဂဏန်းတွေကို တွဲပြီး နှစ်လုံးဂဏန်း ထွက်ပေါ်လာတာ ဖြစ်ပါတယ်။

ထွက်ချိန်များ

မြန်မာ 2D ဟာ နေ့စဉ် (ရုံးပိတ်ရက် မှလွဲ၍) သတ်မှတ်ထားတဲ့ အချိန် နှစ်ကြိမ် ထွက်လေ့ ရှိပါတယ် — မွန်းတည့်ပိုင်းနဲ့ ညနေပိုင်း။ တိကျတဲ့ အချိန်က ရက်အလိုက် အနည်းငယ် ကွဲပြားနိုင်ပါတယ်။

ဘာကြောင့် လူကြိုက်များလဲ

2D ဟာ မြန်မာ့ နေ့စဉ်ဘဝနဲ့ ကွာလို့မရအောင် ဆက်စပ်နေတဲ့ ယဉ်ကျေးမှု ဖြစ်ပါတယ်။ လူအများစုဟာ အိပ်မက်၊ ကံ၊ နံပါတ်တွေနဲ့ ချိတ်ဆက်ပြီး ဂဏန်းကို ခန့်မှန်းကြလေ့ ရှိပါတယ်။ ဒီယဉ်ကျေးမှုဟာ မိသားစု၊ သူငယ်ချင်းတွေကြားမှာ နေ့စဉ် ပြောဆို ဆွေးနွေးစရာ တစ်ခု ဖြစ်နေပါတယ်။
