2D Set နဲ့ Value ဆိုတာ ဘာလဲ?

ကံကြမ္မာ & ဂဏန်း · ဖတ်ရန် ၃ မိနစ်

မြန်မာ 2D ကို စိတ်ဝင်စားသူတွေ မကြာခဏ ကြားရတဲ့ စကားလုံး နှစ်ခုက Set နဲ့ Value ပါ။ ဒီ နှစ်ခုက ဂဏန်း ထွက်ပေါ်ပုံနဲ့ ဆက်စပ်နေပါတယ်။

Set ဆိုတာ

Set ဆိုတာ နိုင်ငံတကာ စတော့ဈေးကွက်ရဲ့ အရောင်းအဝယ် ညွှန်းကိန်း (Index) ကို ဆိုလိုပါတယ်။

Value ဆိုတာ

Value ဆိုတာ ရောင်းဝယ်မှု စုစုပေါင်း တန်ဖိုး ကို ဆိုလိုပါတယ်။

ဂဏန်း ဘယ်လို ထွက်လဲ

Set ရဲ့ နောက်ဆုံး ဂဏန်း တစ်လုံးနဲ့ Value ရဲ့ နောက်ဆုံး ဂဏန်း တစ်လုံးကို တွဲလိုက်ရင် နှစ်လုံးဂဏန်း (2D) ထွက်ပေါ်လာတာ ဖြစ်ပါတယ်။

ဒီတွက်နည်းကို နားလည်ထားရင် 2D ယဉ်ကျေးမှုကို ပိုနက်နက်ရှိုင်းရှိုင်း နားလည်နိုင်ပါတယ်။
